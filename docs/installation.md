# Installation

To start using `vue-email` in any vue project, you just need to install the package:

With NPM:

```bash
$ npm install vue-email
```

With Yarn:

```bash
$ yarn add vue-email
```

With PNPM:

```bash
$ pnpm install vue-email
```
