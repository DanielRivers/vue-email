# Html
A vue html component to wrap emails.

### Usage

```vue
<template>
  <e-html lang="en">
    <!-- ... -->
  </e-html>
</template>

<script setup>
import { EHtml } from 'vue-email';
</script>
```

| Name |  type  |  Default  |        Description        |
| ---- | :---:  |   :---:   | :-----------------------: |
| `lang` | string |     `en`    | The language of the email |
